# **App Name**: Artful Assistant

## Core Features:

- Color Palette Generator: Inputted keywords will generate a palette of colors suitable for art or design, incorporating hue, saturation, and luminosity appropriate to the content.
- Palette Display: A clean display showing color swatches accompanied by RGB hex codes for each.
- Palette Customization: A field allowing users to specify color names or modify the default name created upon generation of the palette.
- Palette Saving: Ability to save generated palettes for later use or reference.
- Custom Brand Palettes: Importing, editing and saving your own palettes based on your brand needs
- Color Application Tool: An AI-powered 'tool' that intelligently suggests how to apply the colors in the generated palette to example designs and layouts.

## Style Guidelines:

- Primary color: Soft lavender (#D0BFFF) to evoke creativity and calm.
- Background color: Light grey (#F0F0F5) to ensure good contrast and readability.
- Accent color: Muted pink (#E6C5FF) to highlight key elements and create a gentle, harmonious feel.
- Body and headline font: 'PT Sans', a modern, humanist sans-serif suitable for a comfortable reading experience.
- Use simple, clear icons for key functions such as saving and generating palettes.
- A clean and structured layout with the color palette prominently displayed, alongside editable color values.
